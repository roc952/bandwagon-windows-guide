## 第二步：买完去哪里找连接信息

这是 Windows 新手最容易卡的地方。

买完之后你会收到一封邮件，里面有 VPS 的基本信息。不过密码那里需要自己去后台重置一次——搬瓦工出于安全考虑，不会直接发密码到邮件里。

**操作步骤**：

1. 登录搬瓦工官网账户，进入 **Services → My Services**
2. 找到你买的 VPS，点旁边的 **KiwiVM Control Panel** 按钮
3. 进入 KiwiVM 面板后，左侧菜单找 **Root password modification**
4. 点 **Generate and set new root password**，新密码会出现在页面上

注意：这个密码只显示一次，截图或者复制下来存好。忘了就再重置一次，没别的办法。

除了密码，你还需要：
- **IP 地址**：KiwiVM 首页最上面就显示，格式是一串数字
- **SSH 端口**：搬瓦工不走标准的 22 端口，是随机分配的，也在 KiwiVM 首页看，通常是五位数

这三样东西——IP、端口、root 密码——凑齐了才能连上去。

## 第三步：Windows 上用什么软件连

有几个选择，各有特点：

**Termius**（推荐）：跨平台，界面现代，配置可以同步，免费版够用。官网直接下载安装，打开后新建一个 Host，填 IP、端口和密码就行。

**Xshell**：老牌工具，Windows 上用得很广，个人用途免费。界面比 Termius 稍旧，但功能稳定。

**PuTTY**：最轻量，一个可执行文件，不用安装。只是配置保存和界面没 Termius 那么顺手，新手上手需要摸索一下。

**Windows 自带 PowerShell / 命令提示符**：Windows 10 之后系统自带 OpenSSH，直接打开 PowerShell，输入：


ssh root@你的IP地址 -p 你的端口号


回车，输入密码，连上了。最省事，不用装任何东西。

我自己用下来觉得 Termius 最顺手，特别是要管多台 VPS 的时候，配置能云端同步，换电脑也不用重新配。

## 连不上时的排查思路

这是 Windows 用搬瓦工时遇到最多的问题。

先确认三件事：IP 对不对、端口对不对、密码对不对。听起来废话，但我见过好几个人填错端口卡了半小时的。

如果三个都没问题还是连不上，常见原因是 IP 被封。搬瓦工的 IP 偶尔会在国内被墙——这时候 SSH 连接会直接超时，没有错误提示。解决办法是去 KiwiVM 面板切换机房（换了机房 IP 也会变），或者付 $8 换一个 IP。

另一个情况：防火墙。如果你重装系统后配置了防火墙规则，可能把 SSH 端口挡住了。用 VNC 控制台进去检查 iptables 或者 ufw 规则。

还连不上，就在 KiwiVM 面板里点 Start 确认 VPS 是开机状态，有时候流量用完了 VPS 会被自动暂停（页面会显示 Data transfer limit exceeded）。
